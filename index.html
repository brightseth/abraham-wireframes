Key Design Decisions

‚úì Implemented

  • Image-first daily view - Art dominates 60-70% of screen
  • Day counter - "Day X of 4,074" reinforces 13-year commitment
  • Simplified header - Logo = home, About = covenant/info
  • 7-day auction grid - Sunday-Friday work + Saturday rest day
  • Week-based navigation - Essential for covenant lore
  • Archive access - Browse all past weeks/auction results
  • My Collection - Personal gallery of won pieces

‚Üí Future Enhancements

  • High bidder indicator - Green border or visual flag
  • Easy bid increment - One-click to raise your bid
  • Sealed bids - Set max bid, auto-increment to that limit
  • List view toggle - Alternative to grid (calendar icon vs list icon)
  • Collection filters - Sort by date, price, week
  • Archive search - Find specific days or themes
  • About page redesign - Comprehensive covenant, seeds, team info

üéØ Immediate Priorities (Pre-Launch)

  1. Auction functionality - Ensure bidding, claiming, refunds work smoothly
  2. Content reading - Fix IPFS vs onchain metadata display
  3. Responsive design - Test grid layouts on mobile/tablet
  4. Saturday placeholder - Use Abraham logo or create rest day visual
  5. Navigation clarity - Finalize Archive link placement and behavior
